# Security

## Reporting vulnerabilities

**Do not** open public issues for security bugs in the hosted platform.

Email **hello@fuseiq.io** with:

- Description and impact
- Steps to reproduce (SDK/CLI issues)
- Your GitHub handle (optional)

We respond within **72 hours** for confirmed reports.

## Scope

| In scope | Out of scope |
|----------|----------------|
| This repo (SDK / CLI / templates) | fuseiq.io hosted platform (report via email) |
| Dependency vulnerabilities in this repo | Social engineering, rate-limit noise |

## Safe harbor

Good-faith security research on **public repos only** is welcome. Do not access other users' workspaces or production APIs without permission.
